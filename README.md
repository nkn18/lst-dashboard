# LST Dashboard

A Dashboard that compares yield and LP opportunities on Bifrost and other LST protocols using data sourced from the DeFiLlama API.

## Features
- 📊 **Real-Time Data:** Compare current yield across LST protocols.
- 📉 **Historical Analysis:** View yield trends over time.
- 💰 **LP Opportunities:** Explore potential liquidity provision rewards.
- 🔗 **Data Source:** All data is sourced from the DeFiLlama API.


## Getting Started

### Clone the Repository
```sh
git clone https://github.com/nkn18/lst-dashboard.git
cd lst-dashboard
```

### Install Dependencies
```sh
npm install
```

### Build the Project
```sh
npm run build
```

### Start the Application
```sh
npm run start
```
